Object-Oriented Programming

  • Is a way of writing computer programs separating the code into objects that have properties and methods.
  • There are four main concepts in OOP: encapsulation, polymorphism, abstraction and inheritance.

Encapsulation

  • The variables of a class will be hidden from other classes, and can be accessed only through the methods of their current class.

Polymorphism

  • The same process can be used for different objects. Is the ability of an object to take on many forms

Inheritance

  • Taking the features of one object then adding some new features. In OOP, this means we can take an object that already exists and inherit all its properties and methods.

Abstraction

  • it ignores the irrelevant details and shows only the required one. It reduces the duplication of code.

What is a ternary operator?

The ternary operator is said to be an operator which takes three arguments. is frequently used as a shortcut for the if statement.
The ternary operator is also called a conditional operator.
