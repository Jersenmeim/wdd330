Chapter 5 (Objects) Notes

  1. Everything is an object in JavaScript.
  2. A JavaScript object is an entity having state and behavior (properties and method)
  3. A regular expression is a sequence of characters that forms a search pattern.

There are 3 ways to create objects.

  1. By object literal
  2. By creating instance of Object directly (using new keyword)
  3. By using an object constructor (using new keyword)

Chapter 6 (The Document Object Model) Notes

  1. The DOM treats everything on a web page as a node.
  2. 2. The Node object is the primary datatype for the entire Document Object Model. It represents a single node in the document tree.

Some important methods to add/remove new DOM elements.

Method Description
append() Inserts content to the end of element(s)
before() Inserts content (new or existing DOM elements) before an element(s)
before() Inserts content (new or existing DOM elements) after an element(s)
prepend() Insert content at the beginning of an element(s) specified by a selector.
remove() Removes element(s) from DOM
replaceAll() Replace target element(s)
wrap() Wrap an HTML structure around each element

Chapter 7 (Events) Notes

  1. Events occur when a user interacts with a web page.
  2. There are many types of event, including mouse events, keyboard events, and touch events.
  3. The JavaScript change event is an event type that gets executed when the focus on an element is changed. The change event of JavaScript inherits all the methods and properties of the Event.
